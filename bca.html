What is BCA?

Understanding the Bachelor of Computer Applications Program

Program Overview

  • Duration: 3 Years (6 Semesters)
  • Level: Undergraduate Degree
  • Field: Computer Applications
  • Mode: Regular/Distance/Online
  • Aim: IT & Software Development

Course Objectives

  • Programming: Learn coding languages
  • Software: Application development
  • Database: Data management skills
  • Web Tech: Internet technologies
  • Networking: Computer networks

Key Skills Developed

  • Problem Solving: Logical thinking
  • Coding: Programming skills
  • Database: SQL & NoSQL
  • Web Dev: Frontend & Backend
  • Team Work: Collaboration

BCA Eligibility Criteria

Requirements for BCA Admission

Educational Qualification

  • 10+2 Pass: Minimum 50% marks
  • Stream: Any stream (PCM/Commerce/Arts)
  • Math: Math in 10+2 preferred
  • Age: No age limit
  • Boards: Recognized board

Additional Requirements

  • Entrance: University exam (if any)
  • English: Basic knowledge required
  • Computer: Basic computer skills
  • Interview: Personal interview (some colleges)
  • Documents: Required certificates

BCA Admission Process

Step-by-Step Guide to BCA Admission

Application Process

  • Registration: Online/Offline forms
  • Documents: 10th, 12th mark sheets
  • Photos: Passport size photos
  • ID Proof: Aadhar, PAN card
  • Fee: Application fee payment

Entrance Exams

  • IPU CET: GGSIPU Delhi
  • SUAT: Sharda University
  • BITSAT: BITS Pilani
  • Christ University: Own exam
  • LPUNEST: LPU University

Selection Process

  • Merit: 12th marks (40-60%)
  • Entrance: Exam scores (30-50%)
  • GD: Group Discussion (10-20%)
  • PI: Personal Interview (10-20%)
  • Final: Composite score
